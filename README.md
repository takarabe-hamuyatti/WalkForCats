# 猫歩き（WalkForCat）
 
歩数を記録するアプリです。猫好きの人のモチベーションが上がるように目標達成率に合わせて猫の表示が変わるようになっています。
 
 
## 使用技術
 
* MVVM architecture
* Room
* Dagger-Hilt
* Coroutine
* kotlin flow
* workmanager
* Navigation
* RecyclerView
* DataBinding
* BindingAdapter
* Service
* retrofit（実装中）
 
 ## やり残したこと
firestoreやAPI通信を盛り込もうとしたが内容的に必要なところがなかったため未実装  
特にfiresotreは右も左もわからない時に使ったきりになっているので次作るアプリにはぜひ導入したい
